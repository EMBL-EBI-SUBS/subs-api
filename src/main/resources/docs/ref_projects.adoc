= Project
:docinfo: shared
:toc: auto

A submission can contain one project.

== Minimal requirements
Fields:

* title, should be at least 50 characters long
* description, should be at least 50 characters long
* releaseDate, in YYYY-MM-DD format, eg. `"2017-12-12"`
